Motor_dc constructor.

Motor_dc constructor. Initilizes cmdc motor (initialize PWM, and hall effect sensors) The PWM frequency is given by the following parameters

Parameters
ppwm: Pointer over the M21m1_pwm object (to handle the pwm signal)
[in]prescalerValue for setting the clock prescaler. Possible values are: PWM_PRESCALER_NONE PWM_PRESCALER_4 PWM_PRESCALER_32 PWM_PRESCALER_256
[in]sourceClockSet the source clock for the PWM module. Possible values are: PWM_SOURCE_CLK_CPU_CLK PWM_SOURCE_CLK_PLL_32MHZ PWM_SOURCE_CLK_PLL_64MHZ
[in]deadTimeNumberCyclesSet the number of cycles for the dead-time. The duration of one cycle is given by the period of the sourceClock divied by the prescaler. (One eleemntary cycle)
